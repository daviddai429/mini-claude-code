## 核心模块详解

### 核心流程 (ReAct Loop)

```
    User Input
        │
        ▼
  ┌─────────────┐      ┌─────────────┐      ┌─────────────┐
  │   Query     │ ───▶ │   Model     │ ───▶ │   Tools     │
  │   Engine    │      │   (LLM)     │      │   Execute   │
  └─────────────┘      └─────────────┘      └─────────────┘
        │                                           │
        │              ┌─────────────┐             │
        └───────────── │   Observe   │ ◀───────────┘
                       │  (结果解析)   │
                       └─────────────┘
                              │
                              ▼
                         [Loop if tool_use]
                              │
                              ▼
                         Final Response
```

### 子 Agent 机制 (Fork)

```
   Parent Agent
        │
        │  1. buildForkedMessages(directive, history)
        │     • 继承完整 system prompt
        │     • 保留所有 tool_use 块
        │     • 注入 <FORK_BOILERPLATE> 指令
        ▼
   ┌─────────────────────────────────────────────────────────────────┐
   │  2. Spawn Child Agent (独立进程)                                │
   │     • 使用 git worktree 隔离文件系统                           │
   │     • permissionMode: 'bubble' (权限传回父 terminal)          │
   │     • maxTurns: 200                                             │
   └─────────────────────────────────────────────────────────────────┘
        │
        │  3. <task-notification> 回调
        ▼
   Parent Agent (合并结果)
```

## 技术栈

| 层级 | 技术 |
|------|------|
| 运行时 | Bun / Node.js |
| 语言 | TypeScript |
| UI | React + Ink (TUI) |
| API | Anthropic SDK (兼容 OpenAI) |
| 工具调用 | 函数调用 (Function Calling) |
| 状态管理 | 内存 + 文件系统 |

## 工具列表 (45+)

### 文件操作
- `Read` - 读取文件
- `Write` - 写入文件
- `Edit` - 编辑文件
- `Glob` - 文件搜索
- `Grep` - 内容搜索

### Shell 执行
- `Bash` - Bash 命令
- `PowerShell` - PowerShell 命令
- `REPL` - 交互式 REPL

### Agent 管理
- `Agent` - 主 Agent 工具
- `Fork Subagent` - 子 agent
- `Resume Agent` - 恢复 agent
- `TaskCreate` / `TaskList` / `TaskUpdate` - 任务 CRUD

### 浏览器/网络
- `WebFetch` - 获取网页
- `WebSearch` - 网络搜索

### MCP 集成
- `MCP` - MCP 服务器
- `McpAuth` - MCP 认证
- `ListMcpResources` / `ReadMcpResource` - MCP 资源

### 其他
- `Skill` - Skills 工具
- `TaskGet` / `TaskOutput` / `TaskStop` - 任务操作
- `Config` - 配置管理
- `ExitPlanMode` / `EnterPlanMode` - 计划模式
- `EnterWorktree` / `ExitWorktree` - Worktree 操作
- `NotebookEdit` - Jupyter 编辑

---

这是 Claude Code 的完整架构，核心是 **ReAct 循环 + 工具系统 + Agent 管理**，特点是模块化程度高、工具丰富、支持 MCP 扩展。
